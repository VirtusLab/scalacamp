Godzina Temat Prelegent
19:00 - 20:00

When things go south

Spark Streaming is a great solution for ingesting data as it arrives, batching it and performing analysis and transformations using the Spark core execution engine. Continuous operation brings about new challenges, like how to ensure no data loss in the face of failures or how to deal with spikes in incoming data. In this presentation I will discuss various fault tolerance and back pressure strategies implemented in Spark streaming.

Let's talk about Scala IDE - new version is coming

Scala IDE, yes this is Eclipse here. But before you decide to stop reading the rest of this short note think about Reactive buzzword. Scala IDE is going to deliver a new kind of programming tools quite suitable for developing reactive applications. No more breakpoints to place when message is received. No more tedious tracing of message history. Let's just debug it! Still not grabbed? What about macros compiling in the same project? Now Scala IDE gives a possibility to compile projects and macros in one click. About these features and more beyond the time horizon is this talk [Yoda said to his padawan].
